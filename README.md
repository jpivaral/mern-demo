# mern-demo

Clone the repo

```
git clone https://github.com/jpivaral/mern-demo.git
```

Go to mern-demo folder

```
cd mern-demo
```

Start mongo db using docker compose
```
docker compose up -d
```
 Go to backend folder
 ```
 cd backend
 ```

 Install dependencies
 ```
npm install
```

Start the project
```
npm start
```
